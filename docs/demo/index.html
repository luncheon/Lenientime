Lenientime

A time input helper.
Use together with your favorite timepicker -- unless the timepicker inhibits manual input.

Usage and Example
Markup
<script src="https://luncheon.github.io/lenientime/umd/lenientime.min.js"></script>
<input data-lenientime placeholder="Press up/down key or input numbers and blur.">
<input data-lenientime="HH:mm:ss.SSS" value="00:00:00.000">
<input data-lenientime="h:_m:_s a">
Result
Features
Complement fuzzy input on focus out.
Increment/decrement on up/down key pressed.
Increment/decrement on mouse wheel.
Move selection range on tab/shift+tab key pressed.
Popup the coolest time picker (Please use your favorite timepicker.)
Clear icon (Please see input-clear-icon.)
